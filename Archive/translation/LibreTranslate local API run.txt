
Run LibreTranslate Container
- cmd, pull the LibreTranslate Docker image:
     ```bash
     docker pull libretranslate/libretranslate
     ```
- run LibreTranslate container locally:
     ```bash
     docker run -d -p 5000:5000 libretranslate/libretranslate
     ```
- This will run LibreTranslate on `http://localhost:5000`.

Run a container restricted to French only (and english)

docker run -d -p 5000:5000 libretranslate/libretranslate --load-only en,fr
